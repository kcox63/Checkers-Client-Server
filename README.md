# Checkers Client-Server Application

Multiplayer checkers game built in Java using a client-server architecture with real-time gameplay, chat, matchmaking, and persistent player records.

## Features

- Real-time multiplayer gameplay
- Public and private messaging
- User login system
- Matchmaking between players
- Persistent win/loss records
- Play Again functionality
- Server administration tools

## Technologies

- Java
- JavaFX
- Maven
- TCP/IP Networking
- Object Serialization

## Project Structure

### CheckersClient
Contains the JavaFX client application, GUI, chat functionality, and gameplay interface.

### CheckersServer
Contains the server, matchmaking system, game state management, and player record persistence.
